# 事件数据接入功能

## 功能概述

为GoldPilot交易看板右侧的信息模块接入真实数据源，包括：
- **经济事件/日历数据**：显示即将发布的重要经济指标
- **市场快讯/新闻**：实时财经快讯信息

---

## 数据源

### 经济事件/日历
- **来源**：Investing.com财经日历
- **网址**：`https://cn.investing.com/economic-calendar/`
- **方式**：爬虫获取
- **更新频率**：每5分钟

### 市场快讯/新闻
- **来源**：东方财富快讯
- **网址**：`https://kuaixun.eastmoney.com/index.html`
- **方式**：爬虫获取
- **更新频率**：每5分钟

---

## API接口

### 后端接口

#### 获取经济日历
```
GET /api/events/calendar?date=2026-05-21
```

**参数**：
- `date`（可选）：日期，格式 YYYY-MM-DD，不传则获取今天

**响应**：
```json
{
  "success": true,
  "data": [
    {
      "date": "2026-05-21",
      "time": "20:30",
      "country": "美国",
      "event": "非农就业人数",
      "importance": 3,
      "actual": "25万",
      "forecast": "20万",
      "previous": "22万"
    }
  ]
}
```

#### 获取市场快讯
```
GET /api/events/news
```

**响应**：
```json
{
  "success": true,
  "data": [
    {
      "time": "20:52",
      "content": "现货黄金短线拉升，突破2390美元/盎司",
      "hot": true
    }
  ]
}
```

## 使用说明

### 启动服务

1. **安装后端依赖**
   ```bash
   cd goldpilot-backend
   npm install cheerio
   npm install -D @types/cheerio
   ```

2. **启动后端**
   ```bash
   npm run dev
   ```

3. **启动前端**
   ```bash
   cd ../goldpilot-frontend
   npm run dev
   ```

### 数据刷新

- **前端自动刷新**：每5分钟自动调用API获取最新数据
- **后端定时任务**：每5分钟执行爬虫预热缓存
- **手动刷新**：刷新浏览器页面

---

## 注意事项

### 反爬虫应对
- 设置合理的User-Agent
- 控制请求频率，避免短时间内大量请求
- 如被封禁，可能需要使用代理IP

### 降级处理
- API失败时返回空数组，不影响页面其他功能
- 错误日志记录，便于排查问题

### 维护
- 爬虫方式可能因目标网站结构变化而失效
- 需要定期检查和更新选择器
- 建议后续考虑使用稳定的数据源API
