# MT4/MT5 真实行情数据接入指南

## 一、MetaAPI 简介

MetaAPI 是一个云服务，可以连接到MT4/MT5经纪商，获取真实的行情数据和交易功能。

官网：https://metaapi.cloud/

## 二、注册账号

1. 访问 https://app.metaapi.cloud/
2. 注册免费账号（免费版提供1个MT4/MT5连接）
3. 获取 API Token

## 三、添加MT4/MT5账户

### 3.1 在MetaAPI控制台添加账户

登录后，进入 "Provisioning" → "New account"，填写：

| 字段 | 值 | 说明 |
|------|-----|------|
| Broker | VTMarkets | 你的经纪商 |
| Server | VTMarkets-Live 8 | 服务器名称 |
| Account | 27238218 | MT4/MT5账号 |
| Password | Abc1234@ | 观摩密码 |
| Platform | mt4 | MT4或MT5 |

### 3.2 获取账户ID

添加成功后，会得到一个账户ID（类似：8c23b6a6-xxxx-xxxx-xxxx-xxxxxxxxxxxx）

## 四、配置环境变量

在 `.env` 文件中添加：

```bash
# MetaAPI配置
METAAPI_TOKEN=你的API_Token
METAAPI_ACCOUNT=你的账户ID
```

## 五、重启后端服务

```bash
cd goldpilot-backend
npm run build
pm2 restart all
```

## 六、测试连接

```bash
# 连接到MT4/MT5
curl -X POST http://localhost:3005/api/metaapi/connect

# 获取实时报价
curl http://localhost:3005/api/metaapi/quote?symbol=XAUUSD

# 获取K线数据
curl http://localhost:3005/api/metaapi/candles?symbol=XAUUSD&timeframe=M1&limit=100

# 获取账户信息
curl http://localhost:3005/api/metaapi/account
```

## 七、API端点说明

| 端点 | 方法 | 说明 |
|------|------|------|
| `/api/metaapi/connect` | POST | 连接到MT4/MT5 |
| `/api/metaapi/disconnect` | POST | 断开连接 |
| `/api/metaapi/quote` | GET | 获取实时报价 |
| `/api/metaapi/candles` | GET | 获取K线数据 |
| `/api/metaapi/account` | GET | 获取账户信息 |

## 八、前端集成

前端可以通过调用这些API获取真实数据：

```typescript
// 获取实时报价
const response = await fetch('http://localhost:3005/api/metaapi/quote?symbol=XAUUSD');
const data = await response.json();
```

## 九、注意事项

1. **免费限制**：免费MetaAPI账户只支持1个MT4/MT5连接
2. **观摩密码**：当前使用的是观摩密码，只能看盘，不能交易
3. **延迟**：MetaAPI可能有1-3秒延迟
4. **稳定性**：如果经纪商断线，需要重新连接

## 十、常见问题

**Q: 连接失败？**
A: 检查账号密码、服务器名称是否正确

**Q: 数据不更新？**
A: 检查经纪商是否在线，MetaAPI是否已连接

**Q: 支持哪些经纪商？**
A: 支持大部分MT4/MT5经纪商，查看：https://metaapi.cloud/brokers/
